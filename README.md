## Session Format 
* Introductions 
* Demos (3 min. each)
* Everyone shares what they can help with and what they need help with.  
* Split into teams and start working.  

## Coworking Standards 
* Everyone has to work on a project, theirs or someone else's 
* Projects need to have goals
